Security Setup — Key Considerations
  • Set up two-factor authentication (2FA) on all exchange accounts
  • Use a unique, strong password for each exchange (password manager recommended)
  • Write down your seed phrase on paper — never digitally (never photo, never cloud, never email)
  • Store seed phrase in at least 2 secure physical locations
  • Never share your seed phrase with anyone — ever
  • Test your hardware wallet restore process before loading significant funds
