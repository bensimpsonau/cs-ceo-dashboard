## 1. Brand Identity

### Mission Statement
Collective Shift empowers Bitcoin and digital asset investors with professional-grade, systematic intelligence — the research, tools, and community that institutional investors have always had, now accessible to the individual.

### Brand Positioning
- **Not a crypto trading service.** CS is a long-term, research-driven membership for serious investors.
- **Bitcoin-native.** Deep conviction in Bitcoin as a monetary asset; not a token-promotion service.
- **No affiliates. No advertising. No token promotions.** Trust-based, member-funded.
- **Institutional grade, individual access.** Fills the gap between what professional funds use and what retail investors can access.

### Target Audience
- Australian and global investors aged 30–60 seeking structured, data-backed crypto exposure.
- Bitcoin believers who want to invest systematically, not speculate emotionally.
- High-earners who value their time and want done-for-you research.
- People frustrated by noise, hype, and influencer-driven content.

### Brand Personality
| ✅ IS | ❌ IS NOT |
|-------|-----------|
| Direct | Corporate / formal |
| Conviction-driven | Wishy-washy / hedged |
| Educational | Preachy or guru-like |
| Data-backed | Hype-driven |
| Peer-to-peer | Top-down / authoritative |
| Calm confidence | Moon-boy / FUD |

## 9. Slide Design Rules

*For creating Collective Shift presentations (investor decks, education slides, webinar decks, pitch decks).*

### Slide Backgrounds
| Slide Type | Background Colour | Notes |
|-----------|------------------|-------|
| Hero / Title | `#02000D` | Darkest black, full bleed |
| Section intro | `#1B1925` | Neutral Darker |
| Content slides | `#FFFFFF` or `#F9F9F9` | Light mode content |
| Highlight / CTA | `#FC5C03` | Blaze Orange — sparingly |
| Data slides | `#02000D` with chart elements | Dark preferred for data |

### Text on Dark Backgrounds
| Element | Colour |
|---------|--------|
| Primary heading | `#FFFFFF` |
| Body text | `#D9D8DA` (Neutral Lighter) or `#B3B2B6` |
| Accent / highlight text | `#FC5C03` (Blaze Orange) or `#0BF4B9` (Turquoise) |
| Subtext / caption | `#807F86` (Neutral) |

### Text on Light Backgrounds
| Element | Colour |
|---------|--------|
| Primary heading | `#02000D` |
| Body text | `#1B1925` or `#4D4C55` |
| Accent | `#FC5C03` (Blaze Orange) or `#2404FC` (Blue) |
| Subtext | `#807F86` |

### Typography for Slides
| Element | Font | Weight | Size |
|---------|------|--------|------|
| Slide title | Plus Jakarta Sans | 600 | 44–72px |
| Section header | Plus Jakarta Sans | 600 | 36px |
| Body text | Barlow | 400–500 | 18–22px |
| Data labels | Barlow | 600–700 | 14–16px |
| Caption | Barlow | 400 | 12–14px |

### Layout Patterns
- **12-column grid** — standard web layout
- **Generous white space** — do not crowd slides
- **Max 3 key points per slide** — data > fluff
- **Orange accent line/divider** — `#FC5C03`, 2–4px height
- **Logo placement:** Top-left corner, Neutral Darkest (`#282D3A`) on light or White on dark

### Do / Don't
| ✅ Do | ❌ Don't |
|-------|---------|
| Bold declarative statements | Bullet point overload |
| One metric per data callout | Multiple competing charts |
| Full-bleed dark hero slides | Bright/colourful slide backgrounds |
| Blaze Orange for single accent | Orange as a background colour on every slide |
| Barlow body text at ≥16px | Small text below 14px |

## 10. Social Media Design Rules

### Platform-Specific Dimensions

| Platform | Format | Size |
|----------|--------|------|
| Instagram Feed Square | Image | 1080 × 1080px |
| Instagram Feed Portrait | Image | 1080 × 1350px |
| Instagram Story / Reel | Video/Image | 1080 × 1920px |
| Twitter/X Post | Image | 1600 × 900px |
| LinkedIn Post | Image | 1200 × 627px |
| LinkedIn Story | Image | 1080 × 1920px |
| Facebook Post | Image | 1200 × 630px |
| YouTube Thumbnail | Image | 1280 × 720px |
| YouTube Banner | Image | 2560 × 1440px |

### Brand Consistency Rules
1. **Always dark.** Default to dark backgrounds (`#02000D` or `#1B1925`). Light-only posts are the exception.
2. **Blaze Orange accent.** Use `#FC5C03` for one highlight element per post (border, callout, icon).
3. **Plus Jakarta Sans for display text.** Barlow for body/supporting text.
4. **No gradients unless intentional.** Stick to solid fills from the colour system.
5. **Logo always present** — bottom-right or top-right watermark at ≥12% opacity if ghosted, or full opacity if prominent.
6. **Minimal text on images.** Prefer headline only; let copy sit in the caption.

### Text Overlay Rules
| Rule | Spec |
|------|------|
| Headline font | Plus Jakarta Sans 600–700 |
| Headline size (Instagram) | 52–72px |
| Body text (if used) | Barlow 400–500, max 22px |
| Text colour on dark | `#FFFFFF` primary, `#FC5C03` accent |
| Text colour on light | `#02000D` primary, `#FC5C03` accent |
| Max words on image | 10–15 words headline only |
| Contrast minimum | 4.5:1 (WCAG AA) |

### Quote Card Formula
```
Background: #02000D or #1B1925
Quote text: #FFFFFF, Plus Jakarta Sans 600, 28–36px
Attribution: Barlow 400, 16px, #807F86
Orange accent element: thin horizontal rule or left-border in #FC5C03
Logo: bottom-right, white
```

### Story / Reel Thumbnail Formula
```
Background: Dark (#02000D)
Bold orange geometric shape or accent stripe
Headline: Plus Jakarta Sans 700, white, large
CTA text (optional): Barlow 500, orange
No more than 3 visual elements total
```
